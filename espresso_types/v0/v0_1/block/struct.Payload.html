pub struct Payload {
    pub(crate) raw_payload: Vec<u8>,
    pub(crate) ns_table: NsTable,
}
Expand description

Raw payload data for an entire block.

A block consists of two sequences of arbitrary bytes:

  • ns_table: namespace table
  • ns_payloads: namespace payloads

Any sequence of bytes is a valid ns_table. Any sequence of bytes is a valid ns_payloads. The contents of ns_table determine how to interpret ns_payload.

§Namespace table

See NsTable for the format of a namespace table.

§Namespace payloads

A concatenation of payload bytes for multiple individual namespaces. Namespace boundaries are dictated by ns_table. See NsPayload for the format of a namespace payload.
